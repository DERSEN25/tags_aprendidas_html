Tags aprendidas

As tag H, servem para definir níveis de titulos, sendo o h1 é o nivel mais alto e o h6 o menor

A tag P serve para definirmos paragrafos

A tag mark é utilizada quando queremos destacar um trecho do conteudo, como se fosse um marcatexto

Usamos a tag i para colocar um texto em italíco, tag strong para deixa-lo em negrito e a u para sublimar o texto

Utilizamos a tag ol para criar listas ordenadas e a ul para criar listas sem ordenação, dentro delas temos que usar a tag filha li para criar o conteúdo de nossa lista

A tag a é utilizada para colocar hyperlinks na pagina, seja para direcionar para outros sites ou para referenciar outra pagina do mesmo site

A tag hr é usada para criar uma barra para dividir uma pagina

A tag sub é usada para posicionar texto abaixo da linha e a tag sup é para posicionar na parte superior

A tag blockquote é usada quando referemos por um texto de referência no nosso html

Tags Novas

A tag font era utilizada para definir a fonte de letra

A tag del serve para representar uma parte que foi excluída do texto

A tag abbr serve para definir uma abreviação no html
